Spam SMS Detection

This is my first task for CODSOFT Machine Learning Internship.

In this project, I created a machine learning model to classify SMS messages as spam or not spam.

Steps:
- Loaded dataset
- Cleaned unwanted columns
- Converted labels into numbers
- Converted text into numerical format using TF-IDF
- Trained model using Naive Bayes
- Tested the model using custom messages

Tools used:
Python, Pandas, Scikit-learn
